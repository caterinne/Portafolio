Webs y App Móvil

Proyecto Web 1 — Sistema de Inventario y Control Financiero

Plataforma web desarrollada para la gestión integral de una florería, permitiendo administrar inventario, registrar ventas y gastos, y generar reportes financieros automáticos. Incluye autenticación segura mediante Firebase Authentication y base de datos en tiempo real con Firestore, utilizando Angular como framework principal.

Proyecto Web 2 — Gestor de Convenios para Instituciones

Aplicación web diseñada para optimizar la gestión de convenios académicos, enfocada en resolver una necesidad del área ORI-UCM, quienes actualmente administran convenios utilizando planillas Excel. El sistema centraliza información, facilita el seguimiento y permite gestionar estados y documentos mediante una API desarrollada en Node.js con base de datos MongoDB, complementado con una interfaz moderna creada en ReactJS.

Proyecto Web 3 — Sistema de Rendición de Viajes (En desarrollo)

Sitio web en desarrollo para digitalizar el proceso de rendición de viajes utilizado por conductores, reemplazando registros manuales en papel. La plataforma permite registrar gastos, viajes y respaldos, sincronizando datos en tiempo real mediante Firestore y manteniendo el acceso seguro con Firebase Authentication. Construido con React + Vite para un rendimiento rápido.

App Móvil — Control de Inventario y Finanzas (En desarrollo)

Aplicación móvil orientada a pequeños vendedores y emprendedores que requieren administrar inventario, registrar ventas, calcular utilidades y organizar sus finanzas. Incluye autenticación con Firebase Auth, almacenamiento de imágenes con Firebase Storage y base de datos en tiempo real con Firestore.
